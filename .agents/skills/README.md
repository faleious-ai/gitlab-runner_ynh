# MAESTRO skills — program v2

The skills operate under the canonical mandate, backlog, state, findings and planner. They do not create a second planning authority.

The normal sequence is specification, pre-build review when applicable, public-seam TDD, backprop on unexpected failure, read-only checks, two-axis review, guardrails and convergence assessment.

Dynamic correction tasks are canonical through `PROGRAM_FINDINGS.json`. Parallel preparation is valid only with distinct workers and isolated workspaces plus verified artifacts and logs. Technical completion is derived from a published SELF-bound receipt.
