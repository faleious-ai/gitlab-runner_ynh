# MAESTRO Backprop v2

Backprop records the command, seam, observed result, causal classification and regression. Reversible defects are either closed in the current task or registered as an actionable finding with a correction task in `PROGRAM_FINDINGS.json`. Questions outside the technical mandate return to the declared authority boundary.
