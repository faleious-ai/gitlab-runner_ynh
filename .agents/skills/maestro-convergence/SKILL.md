# MAESTRO Convergence v2

Convergence is measured by demonstrated claims, reduced failures, resolved findings, stable interfaces and remaining eligible tasks. Repeated causal failure triggers strategy or specification backprop rather than blind retries. Program completion and external checkpoints are accepted only from the planner's semantic result, not from queue emptiness or a single environment failure.
