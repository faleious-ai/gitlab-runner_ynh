# MAESTRO Guardrails v2

Guardrails validate the real Git diff, protected exact paths and prefixes, rename and untracked cases, task-owned paths, receipt identity, evidence placement and sensitive-data absence. Actions outside the technical mandate remain governed by their declared gates. Protected authority files are not modified to make an implementation pass.
