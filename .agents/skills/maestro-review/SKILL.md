# MAESTRO Review v2

The program axis covers objective, eligibility, dependencies, ownership, acceptance, claims, findings and receipt consistency. The engineering axis covers correctness, security, lifecycle, compatibility, rollback, partial failure and simplicity. Both use the actual Git diff and evidence; neither substitutes for the other.
