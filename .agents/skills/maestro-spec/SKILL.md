# MAESTRO Spec v2

A task specification identifies objective, work package, dependencies, repositories, prefix-aware paths, public seam, acceptance, evidence, operation class, reversibility, rollback and receipt conditions. Reversible implementation detail may be clarified locally; protected objectives, program scope and nontechnical consequences remain under Orchestrator authority.
