# MAESTRO TDD v2

Protected objective tests remain Orchestrator-owned. Focused unit, integration, property and regression tests may be added around the public seam. A valid cycle records the failing behavior before the fix, the smallest coherent implementation and the passing objective plus proportional negative and lifecycle checks. A material contract conflict becomes a contract challenge rather than an unreviewed oracle edit.
