# Papéis no Runner v2

- Maestro Diretor: consequências, privilégios, release e irreversibilidade.
- Orquestrador: backlog, acceptance, review e gates.
- Codex: executa tasks Runner elegíveis, coordena lanes, TDD, receipts e push.
- Subagentes: artifacts/logs isolados, sem commit ou edição canônica.

O Executor não altera oracle, não fabrica estado completed e não converte limitação ambiental em sucesso.
