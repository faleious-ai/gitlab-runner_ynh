# Contrato de execução Runner v2

O charter do coordenador autoriza a janela; backlog/planner seleciona a tarefa. Cada task Runner declara dependencies, paths, acceptance, output e rollback. A tarefa fecha com um commit remoto contendo mudança, evidência e receipt SELF. Review pendente não bloqueia dependentes reversíveis; release, registro real, promoção e credenciais permanecem gated.
